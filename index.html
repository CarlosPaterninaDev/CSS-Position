CSS Positions

La propiedad position de CSS especifica cómo un elemento es posicionado en el documento. Las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados.

Propiedades disponibles:

  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

Static: Es la propiedad por defecto. Establecer una posición diferente a esta permite la utilización de otras propiedades tales como: top, bottom, left, right y z-index.

CSS STATIC POSITION

Relative: El objeto se mueve en base al lugar donde se encuentra originalmente.

CSS RELATIVE POSITION

Absolute: El objeto se ubica de manera absoluta con el elemento más cercano que tenga posición relativa o con el body.

CSS ABSOLUTE POSITION

Fixed: El elemento se muestra de manera fija en el viewport.

CSS FIXED z-index: -1 POSITION

Sticky: El elemento se queda de manera fija una vez que aparece en pantalla.
